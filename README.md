# Phillips Hue & Flic Hub For Raspberry Pi
Tested on Raspbian for Raspberry Pi

## Requirements
- default on raspbian setup

## Running

### First time setup and run:
- Start the Daemon by runnnig ```sudo ./flicd -f flic.sqlite3``` in /bin/arm6l
